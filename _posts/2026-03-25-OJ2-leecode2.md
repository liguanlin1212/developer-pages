## 通过引用方法进行排序（技巧）

https://gcnqbk3mr3ut.feishu.cn/docx/HHDadshCnoMN1ExOMwccFHCXnZd

```
void myswap(int &x, int &y)
{
    int temp = x;
    x = y;
    y = temp;
}

// 排序函数（引用）
void sort3(int &a, int &b, int &c)
{
    if (a < b) myswap(a, b);
    if (a < c) myswap(a, c);
    if (b < c) myswap(b, c);
}
```
小技巧：
在这里不需要定义数组，通过复杂的排序方法来完成这道题，
我们通过定义一个交换函数和一个比较函数，通过交换变量的值，而不改变变量的输出顺序来巧妙的实现

## 快速判断单调数列（技巧）

https://gcnqbk3mr3ut.feishu.cn/docx/LWUFdVU4Vo9Q0sx7nrpcLo7PnZF

```
class Solution {
public:
    bool isMonotonic(vector<int>& nums) {
        return is_sorted(nums.begin(), nums.end()) || is_sorted(nums.rbegin(), nums.rend());
    }
};
```
>
> 这里调用了多个函数：
>  1. is_sorted():定义在<algorithm>中，用于判断区间范围是否是升序序列，若是则返回true；
>  2. arr.begin():指向数组头部迭代器；arr.end():指向数组尾部迭代器;
>  3. arr.rbegin()与arr.rend():反向迭代器；
>  4. is_sorted(nums.rbegin(), nums.rend())不能写成 is_sorted(nums.end(), nums.begin())
> 
>  区间不符合要求
>
>  普通数组不能使用begin\end()：
>  若要适应所有数组类型，我们可以引入<iterator>头文件，使用begin/end()实现；
>  
>  通过或运算符||实现快捷判断；

或者：

```
class Solution {
public:
    bool isMonotonic(vector<int>& nums) {
        bool inc = true, dec = true;
        int n = nums.size();
        for (int i = 0; i < n - 1; ++i) {
            if (nums[i] > nums[i + 1]) {
                inc = false;
            }
            if (nums[i] < nums[i + 1]) {
                dec = false;
            }
        }
        return inc || dec;
    }
};
```
手动实现is_sorted；

总结：
1. 快速判断单调数列；
2. 引入<iterator>,来适应所有类型的数组；
