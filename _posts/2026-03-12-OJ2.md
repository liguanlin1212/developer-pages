##  十六进制转十进制：

笔记： https://gcnqbk3mr3ut.feishu.cn/docx/Na0qd3ejpoCgc3xVUf8ctcHdn0E

总结：

1. char[] 不能使用 .size()，应使用 strlen() 或 string。
2. 字符数字需要转换为真实数值，例如 '3' → 3。
3. 十六进制字符 A-F 需要转换为 10-15。
4. 循环边界应使用 < len，避免数组越界。
5. 推荐直接使用 string 处理字符串，代码更安全简洁。
6. 声明数组时，需要指明数组大小

## 矩阵左转

笔记： https://gcnqbk3mr3ut.feishu.cn/docx/UdARd7FywoM58DxVrdjcbkxWn4d

总结：

1. 二维动态数组必须使用 二级指针分配。
2. C++ 数组默认 从 0 开始索引。
3. 矩阵输入通常按 行读取。
4. 二维 new 分配的数组必须 逐行 delete 再 delete 外层指针。
5. 矩阵左旋 90° 的核心输出顺序：
列：m-1 → 0
行：0 → n-1
6. 为了避免重新创建矩阵，可以选择直接打印；

## 二级指针动态分配二维数组

笔记： https://gcnqbk3mr3ut.feishu.cn/docx/UdARd7FywoM58DxVrdjcbkxWn4d

总结：

动态分配流程：

1. 申请行指针数组。
2. 每一行申请列空间。
3. arr[i][j] 使用。
4. 逐行 delete。
5. delete arr。
