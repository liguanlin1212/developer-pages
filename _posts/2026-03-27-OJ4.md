# OJ4 结构体习题及相关技巧

## 分数四则运算（结构体）

https://gcnqbk3mr3ut.feishu.cn/docx/VdfIdza16oRPjaxUwNucFgounub

总结：
1. 最大公约数：辗转相除法（熟练掌握）
```
int getGCD(int a,int b)
{
  if(a<b)swap(a,b);
  return b==0?a:getGCD(b,a%b);
}
```

2. 化简分数：同时除以最大公约数，注意要调整分子分母的正负号，若有负号统一调整到分子上

3. 对于1/2这种输出，需要定义一个ch专门用于接收“/”
```
fl f1,f2;
    char ch;
    cin>>f1.n>>ch>>f1.m;
    cin>>f2.n>>ch>>f2.m;
//f.n f.m是int
```

## 谁是老二（结构体）

https://gcnqbk3mr3ut.feishu.cn/docx/PduhdRttqo6yIYxjthWcNiQInpb

总结：
1. 通过自己定义比较函数cmp():
    依次比较年、月、日，若相同则比较下一级；
    若不相同，则数字小的（年龄大的）排在前面（符合sort()升序排列）
2. 调用sort()函数，会原地按照比较函数调整原数组
```
bool cmp(stu a,stu b)
{
  if(a.year!=b.year)return a.year<b.year;
  if(a.month!=b.month)return a.month<b.month;
  return a.day<b.day;
}

···

sort(arr,arr+t,cmp);
```
